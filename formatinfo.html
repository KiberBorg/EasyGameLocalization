How to use


Easy Localization allows you to bulk-translate your sheet with a simple principle: Each column is for one language. If you want to add a translation into a certain language just put the Language Code in the first row of the next column.

There are two special columns however:

  1. Column B has your main language, that will be used to translate into all the other languages. Like with the other columns you put the language code in the first row and then row for row all the words you want to be translated.
  2. Column A is ignored by the add-on. You don't have to put anything here but you can use it for identifiers in order to reference your translations later.
