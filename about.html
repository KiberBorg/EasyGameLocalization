Easy Game

Localization



Easy Game Localization is an add-on created to simplify game localization with Google spreadsheets. It expects your spreadsheet to be organized in this way: and translates it fully. It is a fork of this add-on which may serve your purposes better if you don't want to reorganize your spreadsheet. It uses the Google Translate API and supports more than 100 languages.


You can find and contribute to the source code on GitHub.
